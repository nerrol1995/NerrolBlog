数组

返回数组中的最大值
const arrayMax = arr => Math.max(...arr);
console.log(arrayMax([10, 1, 5]));//10


返回数组中的最小值
const arrayMin = arr => Math.min(...arr);
console.log(arrayMin([10, 1, 5]));//10
